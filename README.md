# 基于 KuaiRand 的短视频用户行为与推荐策略分析

## 项目简介

本项目基于 KuaiRand-Pure 短视频推荐数据，围绕用户行为、
业务指标、异常归因、A/B实验、机器学习和因果推断，
构建完整的数据科学业务分析流程。

## 数据来源

数据来源：[KuaiRand](https://github.com/chongminggao/KuaiRand)

原始数据不上传至本仓库，需从官方渠道下载。
